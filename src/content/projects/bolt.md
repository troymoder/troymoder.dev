Bolt is a general build system that uses Starlark as its configuration
language. It is in the very early stages of development and is not yet ready
for production use. This project is a learning project for me. The goals are
to be a general purpose build system that can be easily extended, understood
and fully compatible with the remote build protocol. Inspired by
[Bazel](https://bazel.build) and [Buck2](https://buck2.build).
