## Getting Started

This theme is inspired by the [Hugo Archie theme](https://github.com/athul/archie). It features a minimal, markdown-ish aesthetic.

### Features

The theme includes:

- Tags taxonomy
- Syntax highlighting
- Social links in the footer
- Clean, readable typography
